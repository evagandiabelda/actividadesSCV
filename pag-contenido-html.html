Cómo modificar contenido HTML"

JavaScript es un lenguaje de programación ampliamente utilizado en el desarrollo web, y una de sus capacidades más notables es la capacidad de modificar el contenido HTML de una página en tiempo real. Esto permite crear experiencias interactivas y dinámicas para los usuarios.

Un ejemplo sencillo de cómo JavaScript puede cambiar el contenido HTML es el uso del método getElementById. En el código de ejemplo a continuación, tenemos un párrafo HTML identificado con el id="demo" y un botón que, al hacer clic en él, cambiará el contenido del párrafo:

What Can JavaScript Do?

JavaScript can change HTML content.

Cuando haces clic en el botón, el JavaScript toma el elemento con el id="demo" y modifica su contenido utilizando la propiedad innerHTML. En este caso, el contenido del párrafo se actualiza a "Hello JavaScript!".

Este es solo un ejemplo simple de lo que JavaScript puede lograr en la manipulación del contenido HTML. Puedes utilizar JavaScript para realizar tareas más complejas, como cambiar estilos, agregar o eliminar elementos, gestionar formularios y mucho más, lo que abre un mundo de posibilidades en la creación de sitios web dinámicos y atractivos.
