Cómo modificar atributos HTML"

JavaScript ofrece una potente capacidad para modificar atributos de elementos HTML en tiempo real. Esto es esencial para interactuar con tu página web y cambiar dinámicamente ciertos aspectos de tus elementos. Una tarea común es la modificación de atributos, como el atributo src de una imagen o el atributo href de un enlace.

Cambiar la Fuente de una Imagen

En este ejemplo, tenemos una imagen con el atributo src inicialmente configurado en "imagen-1.jpg". Cuando el usuario hace clic en el botón, se llama a la función cambiarImagen(), que utiliza JavaScript para cambiar el valor del atributo src de la imagen a "imagen-2.jpg". Como resultado, la imagen se actualiza con una nueva fuente.

Este es solo un ejemplo simple de cómo JavaScript puede modificar atributos HTML. Puedes aplicar esta técnica para ajustar enlaces, imágenes, estilos y otros atributos en función de las interacciones del usuario, lo que te brinda un mayor control sobre la apariencia y el comportamiento de tu página web.
